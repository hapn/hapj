Class: ajax

hapj. ajax

new ajax()

利用XmlHttpRequest发出请求，并进行相应处理的组件。接口实现和jQuery的很类似，但是代码更轻巧，且内置对ajax请求串行化的支持。
Source:
  • hapj.js, line 2207
Example
 
示例
// 队列方法1
for(var i = 0; i < 1000; i++) {
	hapj.ajax({
		url:'/foo.php?i=' + i,
		method:'post',
		queue:true,
		success:function(ret) {
			alert(ret);
		}
	});
}
// 开始处理队列请求
hapj.ajax.endQueue();

// 队列方法2
hapj.ajax.startQueue();
for(var i = 0; i < 1000; i++) {
	hapj.ajax({
		url:'/foo.php?i=' + i,
		method:'post',
		success:function(ret) {
			alert(ret);
		}
	});
}
// 开始处理队列请求
hapj.ajax.endQueue();

hapj. ajax

new ajax(options) → {Object}

Parameters:
Name Type Description
options Object 配置参数，包括如下参数
url
网址
type
网络请求方法，如get、post等，默认为get。
data
发送的数据，如果为get方法，将会串行化数据附加到url里边作为参数发送。如果为post方法，则直接将数据post到服务器。可以是字符串或对象。
dataType
接收的数据的格式，可以是html、json等。默认为json
success(ret)
请求成功后将调用的方法。
error(code)
请求失败后将调用的方法。其中code为http状态码，如404等。
queue
是否启用队列支持。默认为false。
async
是否发出异步请求。默认为true。如果为false，则只有等到页面请求完毕后才会执行后面的代码。
