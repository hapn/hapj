jQuery.fn. switchable

new jQuery.fn.switchable()

使几组相互联动的元素能互相切换
详见例子

Example
 
var sw = $('#cycle').switchable({
	method:'hover',
	cycleTime: 2000,
	map:function(){
		return $(this.getAttribute('href').substr(1) + 'List')[0];
	},
	trigger: function(triggers) {
		triggers.css('color', '');
		this.style.color = 'red';
	},
	target: function(targets) {
		targets.css('display', 'none');
		this.style.display = 'block';
	}
});
sw.rand();

 
$('#sortUl').switchable({
	tag:'li',
	method:'scroll', 
	map:$('#relateUl > div'),
	trigger:function(ts) {
		ts.removeClass('on');
		this.className = 'on';
	},
	stopTop:-20, // 停留在顶部时的上边距，仅在scroll模式下启用
	disableClick:false // 是否禁用click，默认为false，表示点击trigger的时候会自动滚动到对应位置
});

jQuery.fn. switchable

new jQuery.fn.switchable(opt)

switchable的构造函数
Parameters:
Name Type Description
opt Object 选项
method
触发的方法 可选值为hover、click、scroll，默认为click
map
实现将触发元素关联到目标元素的映射函数
trigger
触发后触发元素做出的响应函数
target
触发后目标元素做出的响应函数
cycleTime
自动循环的时间，默认为0
tag
使用当前元素的指定标签的元素作为触发元素
stopTop
停留在顶部时的上边距，仅在scroll模式下启用
endTop
点击后定位到触发元素时偏移的距离，仅在scroll模式下启用
disableClick
是否禁用click，默认为false，表示点击trigger的时候会自动滚动到对应位置
