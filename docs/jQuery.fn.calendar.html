Class: calendar

jQuery.fn. calendar

new jQuery.fn.calendar(options)

这个组件主要用来显示日历
Parameters:
Name Type Description
options Object 参数，包含如下参数：
input
日历绑定的input控件，指定此对象后日历对象会和input控件形成交互
className
日历控件的样式名称，默认为cal
format
日期格式，用Y、m、d、H、i、s分别代表年、月、日、时、分、秒。默认为Y-m-d。
startDate
选择时间区间-开始日期[可选]，格式：Y-m-d。
endDate
选择时间区间-结束日期[可选]，格式：Y-m-d。
onSelect:function(y, m, d, h, i, s)
当选择日期后调用的事件处理函数
y 年份
m 月份
d 日期
h 时钟
i 分钟
s 秒钟
