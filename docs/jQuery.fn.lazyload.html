Class: lazyload

jQuery.fn. lazyload

new lazyload(options)

使元素内的图片元素能延迟加载
Parameters:
Name Type Description
options Object 参数，具体如下：
lazyTag
延迟加载的标识属性，默认为hll
onComplete:Function
当元素内所有图片加载完成时调用的方法
minLoadOffset
开始加载图片的最小时间间隔，单位：ms，默认为30
minExecTime
每次执行载入图片的最小间隔，单位：ms，默认为300
docTopRange
文档顶部距离，单位：px，默认为当前窗口高度的1/4
docBottomRange
文档顶部距离，单位：px，默认为当前浏览器窗口高度的3/4
loadSrc
等待时替换的图片
