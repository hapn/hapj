Class: selectable

jQuery.fn. selectable

new jQuery.fn.selectable(options)

主要用来将普通的下拉框组件(或ul)转化为可以自定义样式的下拉框组件。绑定到原有下拉框的事件也会起作用。
Parameters:
Name Type Description
options 配置参数，目前支持的有：
selectedClassName:string
表示选中时的样式名称，默认为on
showEvent:string
用来显示菜单的事件，可选值为mouseover、mouseenter、click，默认为click。
pack:function(o)
用来设定选择选项后显示的文字
被选中的option或被选中的li
hideDelayTime:Int
隐藏延迟的时间，只有showEvent为mouseover或mouseenter才有效，如果该值小于0，则不会自动掩藏
onShow:function()
显示时调用的事件
onHide:function()
隐藏时调用的方法
onChange:function()
更改时调用的方法
