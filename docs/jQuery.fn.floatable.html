Class: floatable

jQuery.fn. floatable

new floatable()

使元素能相对另一个元素浮动起来。此代码可以使元素在整个页面居中、或浮动在页面左上角，右上角等
Version:
  • 1.0
Author:
  • ronnie<dengxiaolong@jiehun.com.cn>
Source:
  • ui/hapj.floatable.js, line 1
Example
// 设置divMenu元素处于linkMenu元素的左上角
$('#divMenu').floatable($('#linkMenu')).left().top();
