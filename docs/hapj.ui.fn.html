Class: fn

hapj.ui. fn

new hapj.ui.fn()

通过hapj.ui.id和hapj.ui.tag等方法，会返回一个这样的实例，用这个实例来进行相关的操作。hapj.ui.node对象支持push操作，有length对象，能像数组一样通过下标访问。 hapj.ui.node的原型为hapj.ui.fn，所以hapj.ui.fn的所有属性都能通过hapj.ui.node的实例调用到。
